Executive Summary

Overall accessibility is good - the main issues encountered were with missing alt-tags, titles, or tab order issues that can be quickly corrected.

Scope of the Evaluation

Website name
housing.appstate.edu
Scope of the website
Review includes the homepage of the University Housing site along with a maximum of 8 other pages chosen to provide the greatest range of content types throughout the site. 1 random page will also be included in the review. Content types include the following: tables, accordions, collapsible elements, embedded media, tab panes/interfaces, modals, and cards.
WCAG Version
2.1
Conformance target
A
Accessibility support baseline
Internet Explorer, Microsoft Edge, Opera, Safari, Google Chrome, Mozilla Firefox

Observations:

Overall, while the tab order is present and makes logical sense, it may be a bit of an issue that the menu links are numerous enough to push the tab order of the main body content into the 90+ range on a number of the sample pages. Also, instances of tabbed navigation within tab panes for some content has an issue with some of the content being bypassed in the tab order and needs to be corrected.

Homepage

No observations added

Raven Rocks

Tab order of the tabbed navigation near the bottom of the page - it cycles through the tabs, then back to the content of the first tab pane, then bypasses the content of the remaining tab panes (although, hitting 'Enter' on a subsequent tab will allow for the tab order to be transferred to the active pane - but still does not allow for navigation to content of other tab panes). Also, tab order is affected if user opens one of the collapsible elements.
