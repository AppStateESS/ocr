ADA Report List

About the Evaluation

We are using the WCAG-EM Report Tool to evaluate the websites. To help test the website we are using the Accessibility Testing Tool ANDI. Through ANDI we can see what what is being output, focusable elemtnts, graphics/images, links/buttons, structures, tables, color contrast, and Iframes. This tools gives accessibility alerts that the user can then manually go through to determine if it is an issue.

Our baseline browser support is the following: Microsoft Edge, Opera, Safari, Google Chrome, Mozilla Firefox

Scope of the Evaluation

Review includes the homepage along with pages chosen to provide the greatest range of content types throughout the site. At least one random page will also be included in the review. Content types include the following: tables, accordions, collapsible elements, embedded media, tab panes/interfaces, modals, and cards. Report will include an overview of all errors identified with examples where needed.
