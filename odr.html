ODR Home

One possible issue - the "Campus News" bold text in the first paragraph should possibly be tagged as a header with the accompanying text included in a <p> below it. Currently it is just part of the <p> element itself and may be confusing for screen reader users.

About ODR

Staff names should probably be tagged as a smaller <h> level rather than simply strong text within a paragraph.

Disability Definitions

Workable as it is, but maybe using a combination of header tags and/or a definition list would be more friendly for screen readers.

Disability Laws

Similar to the other pages, it is likely ok as is but would benefit from the use of header tags rather than <strong> text within paragraphs to better section text.
