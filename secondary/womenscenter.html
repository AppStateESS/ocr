Entire sample

Observations:

Overall, there is a recurring theme of decorative images with vague titles, etc. not hidden from assistive technologies. Several instances of images containing text with no text alternatives available. Rated 'passed' due to more pages passing than not; however there are issues that need to be addressed.

Home Page

Story images have no alt tags or titles, but captions are allowed in this module. Any image without captions is assumed to be decorative. Some titles are vague and not very descriptive and should be changed. Decorative images should be hidden via null alt tag or aria-hidden attribute. Several linked images in right nav block without descriptive titles or alt-tags (appears throughout site, so noted here - not considered for rating of individual home page)

Vagina Monologues

Some titles are vague and should be changed. Images contain text information (dates, times, etc.) that is not presented via text alternative.

Volunteering

Linked image without appropriate markup for assistive technologies

Resources

Links should have appropriate aria-labels and be identified as external where necessary.

Pride Week

Page consists of one image with text with no alternate text available.

Women's History

Page consists of multiple images with text information without any alternate text. Titles are non-descriptive.

About

No non-text content

Body Positive

Some linked images that do not have appropriate descriptive text

Grad Assistant

Decorative image should be hidden from assistive technology

Domestic Violence

Page consists of nothing but images with text without any alternative text options available
